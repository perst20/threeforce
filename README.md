# triforce

Find out if you can triforce

# Dependencies

- CMake >= 3.17
- gcc 10.2.0
- gcc-libs 10.2.0
- X11 - libx11 1.7.0
- OpenGL 3.3 - mesa 21.0.1
- glew 2.2.0

# Build

```
git clone https://github.com/perst20/triforce
cd triforce && mkdir build && cd build
cmake ..
cmake --build .
./triforce
```

Tested on Manjaro Linux 21.0.1
